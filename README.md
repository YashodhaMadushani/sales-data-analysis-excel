# sales-data-analysis-excel

## Description
This project demonstrates a beginner-level sales data analysis using Microsoft Excel. The analysis include calculating total sales, average sales, and product wise performance using Excel formulas

## Tools Used
Microsoft Excel

## Features
- Calculated total sales using formulas
- Used SUM, AVERAGE, COUNT, and SUMIF functions
- Created column chart to compare sales by product
- Created pie chart to show product contribution

## Learning Outcome
This project helped me understand basic data analysis concepts, data summarization, and visualization using Excel.
